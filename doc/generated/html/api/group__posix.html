Detailed Description

Xenomai POSIX skin is an implementation of a small subset of the Single Unix specification over Xenomai generic RTOS core.

The following table gives equivalence between native API services and POSIX services.

Native API services POSIX API services
Alarm services. Clocks and timers services.
Condition variable services. Condition variables services.
Event flag group services. no direct equivalence,
see Condition variables services.
Memory heap services. Shared memory services.
Interrupt management services. Interruptions management services.
Mutex services. Mutex services.
Message pipe services. no direct equivalence,
see Message queues services.
Message queue services. Message queues services.
Counting semaphore services. Semaphores services.
Task management services. Threads management services.
Timer management services. Clocks and timers services.
