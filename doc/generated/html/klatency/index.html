NAME

klatency - Xenomai kernel latency test

1. SYNOPSIS

klatency [ options ]

2. DESCRIPTION

klatency is part of the Xenomai test suite. It is a latency measurement tool dedicated to kernel-based activities. The system must run a suitable Xenomai enabled kernel with the respective module (klat_pipe).

3. OPTIONS

klatency accepts the following options

-l <data-lines per header>

default=21, 0 to suppress headers

-T <test_duration_seconds>

default=0, so ^C to end

-q

supresses RTD, RTH lines if -T is used

-P <rt_pipe_no>

number of testing pipe, default=auto
